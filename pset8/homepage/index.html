CSS

This is a project for CS50 problem set numer 8. The goal of this webpage is to allow user preview css code and make it easier to adjust it for user's needs. It's an attempt to recreate some functionality of https://html-css-js.com/css/. To create first page - border.html I used box model, for the second and third - gradient.html and shadows.html I used flexbox model.

Border editor

Allows user to edit different properties of border for example:
border-radius: 26px 30px 28px 17px;
border: 2px dashed rgb(68, 96, 124);

Gradient editor

Allows user to adjust gradient color and style for example:
background-image: linear-gradient(to right, #e66465 0%, #3f87a6 50%, #9198e5 100%);

Text shadow editor

Allows user create shadow for text and adjust it for example:
text-shadow: rgba(206, 89, 55, 0.69) -3px 16px 2px, rgba(206, 89, 55, 0.69) -3px 13px 2px, rgba(206, 89, 55, 0.69) -3px 9px 2px, rgba(206, 89, 55, 0.69) -3px 7px 2px, rgba(206, 89, 55, 0.69) -3px 3px 2px, rgba(206, 89, 55, 0.69) -3px 16px 2px;
